Description

Multilanguage Articles did not get added to the sitemap. Therefore you will not rank for your translated blog content. Example: Create a blog in your primary language and translate the blog article into a different language.

Context

Version: Odoo V15 Database URL: example.odoo.com Existing customization: No App-Name: TBC

Business need

You cannot generate visitors for your translated blog articles because SEO will not know that this article exists in the first place.

Functional specification

After installing the App, a new sitemap with the existing blog articles gets created based on the configuration in the scheduled actions. Go to the scheduled actions and configure the update interval. After that, activate the schedule action . Finally, run the sitemap manually for the first time and check your new generated sitemap: /sitemap.xml
